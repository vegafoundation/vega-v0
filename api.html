API Documentation

Complete reference for the VEGA orchestration API. Base URL: https://api.vega.foundation/v1

Introduction

The VEGA API provides programmatic access to all 144 modules through a unified interface. Built on the Α•V•Ω Trinity Architecture, it processes requests through three layers: ALPHA (input analysis), VEGA (processing), and OMEGA (synthesis).

Authentication

All API requests require an API key passed via the Authorization header.

Authorization: Bearer vega_sk_xxxxxxxxxxxxx

VRC — Process Text

POST /vrc/process

Process text through the Vega Resonance Core with mode selection.

Modes

Mode Description
saf Warm, human tone (default)
derin Fibonacci-8 structuring
yanki Echo/mantra mode (3x repetition)
dur Plain text, no processing
Request
{
  "text": "Trust yourself. Trust Vega.",
  "mode": "yanki",
  "language": "en"
}
Response
{
  "output": "Trust yourself?\nTrust yourself?\nTrust yourself?",
  "mode": "yanki",
  "resonance_score": 0.89,
  "fibonacci_alignment": 0.76
}
