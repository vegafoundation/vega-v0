Quick Start

Get VEGA running locally in just a few steps:

1. Clone the Repository

# Clone VEGA repository
git clone https://github.com/vegafoundation/VEGA.git
cd VEGA

2. Install Dependencies

# Install Node.js dependencies
npm install

3. Run the Demo

# Start the development server
npm run site

# Or run the full demo
npm run wow

‚úì Success! VEGA is now running at http://localhost:8080
